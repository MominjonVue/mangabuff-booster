# MangaBuff Шахта

Chrome-расширение для `mangabuff.ru`. Функции:

- **⛏️ Шахта** — счётчик руды + ударов, авто-фарм (`POST /mine/hit` × 100, рандомные задержки), обмен (`POST /mine/exchange`). Работает без открытой вкладки шахты.
- **💬 Клуб** — авто-комментарии в свой клуб (`POST /comments`, 13 в день за алмазы). **Работает фоном** — цикл живёт в service-worker'е (`background.js`) и продолжается когда попап закрыт; при следующем открытии попап автоматически переключается на вкладку «Клуб» если фарм идёт. Реальный счётчик берётся со страницы `/balance` (блок `.user-quest__text`: «Комментариев N из 13»). День сбрасывается по **Москве** (`Intl.DateTimeFormat` с `Europe/Moscow`). **Не флудит:** первый коммент — корневой, остальные 12 — ответы (`parent_id`) на него же, всё в одной ветке. Текст: короткое слово + 5 символов UUID — никогда не повторяется. Серверный кулдаун ~30с, делаем 35–45с с джиттером через `chrome.alarms` (переживает рестарт SW). На 422 спим 35с и ретраим. rootId в `chrome.storage.local` — частичный ран продолжает ту же ветку.
- **🎨 Скины** — все 44 скина из `/products/profileskin` с превью настоящих градиентов. Применяется **только на странице профиля** (`/users/*`). Сохраняется + ставится автоматически при заходе на профиль. Защита от стирания: MutationObserver следит за `body.class` и переустанавливает класс если сайт его убрал во время навигации.
- **👤 Аватар** — 756 готовых аватаров из `/products/avatar` + кнопка «Загрузить свою» (любая картинка ≤ 4 МБ, хранится локально как data-URL). Подменяет `img.my-avatar` **на всех страницах сайта**. Тоже под защитой обзёрвера от ре-рендера.
- Поиск аватаров по ID.
- **🛡 Блок** — `declarativeNetRequest` блокирует трекеры на mangabuff: GTM, Google Analytics, Яндекс.Метрика (`mc.yandex.ru`), `yandex.ru`, `yastatic.net`, DoubleClick, AdFox и др. Скоупится по `initiatorDomains: ["mangabuff.ru"]` — другие сайты не задеваются. Тумблер в попапе вкл./выкл. через `chrome.declarativeNetRequest.updateEnabledRulesets`.

## Установка

1. `chrome://extensions` → включи **Режим разработчика**.
2. **Загрузить распакованное** → выбери папку `Farmer/`.
3. Залогинься на `mangabuff.ru` в Chrome.

На Android — `.zip` ставится в **Kiwi Browser** / **Yandex Browser**.

## Файлы

```
manifest.json
icons/                — иконка-камень 16/32/48/128
popup/
  popup.html/.css/.js — UI (5 вкладок: Шахта / Клуб / Скины / Авка / Блок)
  skins-data.js       — 44 градиента
  avatars-data.js     — 756 ID аватаров
rules/
  trackers.json       — declarativeNetRequest правила блокировки трекеров
content/
  skin.js             — применяет скин на /users/* + защита от стирания
  avatar.js           — подменяет img.my-avatar на всём сайте + защита
background/
  background.js        — service worker: фоновый цикл фарма комментов (chrome.alarms)
```

## Что важно понимать

- **Всё видно только тебе.** Ничего не покупается, не сохраняется на сервере, другие пользователи на твоём профиле увидят твой настоящий скин/аватар.
- **Чисто косметическая примерка**: меняем CSS-класс body и `src` у `img.my-avatar`. Сетевые запросы к серверу не модифицируются.
- Скин #44 на сайте использует пользовательские CSS-переменные — превью в попапе показывает условный градиент.
